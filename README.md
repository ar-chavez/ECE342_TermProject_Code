This repository serves as the hub for my Junior Design 2 (ECE342) term project at Oregon State University. My role in this project was designing, validating, and verifying 3/7 blocks 
for a custome detection timer. I worked on the object detector, display, and housing blocks. The team consisted of myself and two other ECE students at OSU. 

The code in this repository was used to verify or support my design choices. The code is written in C++ and runs on an ESP32 using Arduino IDE.
